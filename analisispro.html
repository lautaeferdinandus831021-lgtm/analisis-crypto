📖 Cara Install

  1. Download file .mq4 atau .mq5
  2. Buka MetaTrader di PC/Android
  3. File → Open Data Folder
  4. Masuk ke folder MQL4/5 → Experts
  5. Paste file yang sudah didownload
  6. Compile EA (tekan F7 di MetaEditor)
  7. Drag dan drop EA ke chart
  8. Aktifkan AutoTrading dan setting parameter
